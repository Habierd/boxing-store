# boxing-store

Sitio web estático de una tienda de artículos de boxeo, desarrollado con HTML, CSS y JavaScript, publicado con GitHub Pages y conectado al dominio `boxingstore.cc`.

## Tecnologías
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Estructura del proyecto

```txt
boxing-store/
├── assets/
│   ├── images/
│   │   ├── products/
│   │   └── qr/
│   └── logo/
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── index.html
```

## Objetivo
Desarrollar una tienda web estática con una estructura limpia, profesional y fácil de mantener, enfocada en la venta y exhibición de productos de boxeo.

## Estado del proyecto
En desarrollo.

## Publicación
El proyecto será publicado con GitHub Pages y enlazado al dominio personalizado:

[https://boxingstore.cc](https://boxingstore.cc)
